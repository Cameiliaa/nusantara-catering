# Nusantara Catering System

Sistem manajemen katering berbasis web yang dirancang untuk membantu pengelolaan operasional bisnis katering secara terintegrasi, mulai dari pemesanan pelanggan, manajemen menu, produksi, inventaris bahan baku, transaksi, hingga laporan keuangan.

## Fitur Utama

### Dashboard

* Ringkasan aktivitas sistem secara realtime
* Statistik pesanan
* Informasi produksi
* Ringkasan inventaris

### Manajemen Pesanan

* Tambah pesanan baru
* Edit pesanan
* Kelola status pesanan
* Perhitungan total otomatis

### Katalog Menu

* Kelola menu katering
* Kelola harga menu
* Kelola resep menu
* Hubungkan menu dengan bahan baku

### Manajemen Pelanggan

* Tambah pelanggan
* Edit data pelanggan
* Riwayat pelanggan

### Produksi

* Perencanaan produksi
* Penugasan staf dapur
* Monitoring status produksi
* Kebutuhan bahan baku otomatis

### Inventaris

* Manajemen stok bahan baku
* Manajemen kemasan
* Monitoring stok minimum
* Riwayat pergerakan stok

### Transaksi

* Pencatatan pemasukan
* Pencatatan pengeluaran
* Saldo bisnis realtime

### Laporan Keuangan

* Analisis pendapatan
* Analisis pengeluaran
* Saldo bersih
* Riwayat laporan bulanan

### Profil Perusahaan

* Informasi perusahaan
* Kontak perusahaan
* Lokasi operasional
* Jam operasional
* Panduan penggunaan sistem

### Pengaturan

* Pengelolaan informasi perusahaan
* Pengelolaan jam operasional
* Kontak bantuan sistem
* Informasi sistem

## Instalasi

Clone repository:

```bash
git clone <repository-url>
```

Masuk ke folder project:

```bash
cd nusantara-catering
```

Install dependency:

```bash
npm install
```

Jalankan aplikasi:

```bash
npm run dev
```

Build production:

```bash
npm run build
```
